O título de uma página pode ser criada usando < h1 >
As tags h, elas podem ir de 1 até 6, diminuindo o tamanho dos títulos;
< h1, h2, h3 ... h6 >

Aprendendo estrutura de HTML

Aqui fica os Parágrafos de uma Página, conteúdos, etc!!

Para criar parágrafos utiliza a tag < p >
O p vem de Parágrafo;

Para criar uma quebra de linha, deve-se utilizar o < br >,
O br vem de Break, quebra;

Para criar um negrito, deve-se utilizar o < strong >
O strong vem de Forte, Também dá pra utilizar esse style no CSS usando o comando font-weight;

Para criar um Itálico, deve-se utilizar o comando < em >
O em, representa um comando no CSS também, o font-style: italic;

Para inserir uma imagem, deve-se utilizar a tag > img src="caminho" alt="descrição visual" width="largura em px" < ;
